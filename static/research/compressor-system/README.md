# Compressor system research assets

This directory stores evidence-backed assets for the compressor system map.

- `overview/`: deterministic system architecture figures.
- `concepts/`: concept mechanism figures and worked examples.
- `routes/`: format and compressor route figures, grouped by implementation.
- `benchmarks/`: raw CSV/JSON and plots generated from those records only.
- `documents/`: specifications or project documents with source/version/license notes.
- `audio/`: optional narration; no autoplay and no placeholder audio.

Do not place invented benchmark curves or generated bitstream diagrams here. Every
benchmark plot must be reproducible from a raw result file, and every format diagram
must cite a public specification or a versioned implementation source.
